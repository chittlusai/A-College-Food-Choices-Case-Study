# Comprehensive Analysis and Dietary Strategies with Tableau

## Project Description
This project analyzes college students' food choices using Tableau dashboards and Flask integration. It provides interactive visualizations to understand dietary patterns, health categories, breakfast habits, exercise frequency, and food preferences.

## Technologies Used
- Tableau Public
- Python
- Flask
- HTML
- CSS

## Features
- Interactive Dashboard
- Story Visualization
- KPI Cards
- Food Choice Analysis
- Health Category Analysis
- Tableau Public Integration
